Linux containers

Rootnode is trying to merge the best parts of shared hosting and VPS. We are proud to introduce Linux containers or LXC, a secure and stable platform, to fulfill your deepest desires :)

The container is a preconfigured, fully-fledged Linux system with great user isolation, guranteed resources and unrestricted shell access to run any software you like.

Web applications

The container is an ideal native environment for web application hosting in Ruby, Python and PHP. It runs Apache2 web server with Nginx as a reverse proxy in front giving you unlimited configuration options.

We are working on the Fastweb, a dedicated, high-available web app platform to process millions of request in the blink of an eye.

No limits

Our users get Rootnode account to do things quicker and easier. Because we don't want to make your work harder, we have a very flexible approach to the limits.

That means no transfer limits, expansible drive space, no domain limits, as long as you behave rationally. If you need more resources, ask for more and very likely you will get more.

Satan inside

Rootnode users do things differently. Therefore, your account is equipped with Satan, the best command line service manager developed by us.

With fancy syntax and fast execution, it allows you to configure basic services such as domains, virtual hosts, mailboxes, ftp accounts and do many other tasks.
