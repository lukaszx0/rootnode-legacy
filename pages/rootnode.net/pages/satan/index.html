Satan

Satan is the best, tailor-made service manager out there with command line interface. No clicking, no time wasting, no hassle. Keep it simple, stupid.

Available modules

Satan provides the following modules.

  • DNS to manage domanis
  • Mail to manage mail accounts and aliases
  • Vhost to manage web sites / virtual hosts
  • MySQL to manage MySQL databases and users
  • FTP to manage FTP accounts

Architecture

Satan is divided into following parts.

NameDescription
ServerDaemon awaiting for connection from clients. It reads commands in plain-text or JSON format. Server requires authentication.
ClientUser program to send commands to Satan from shell. You could also us telnet or nc but using client is more convenient.
AgentAgent performs required server-side action. Usually by adding entries to database and reloading services.
ExecutorExecutor runs necessary action inside container with user privileges.
